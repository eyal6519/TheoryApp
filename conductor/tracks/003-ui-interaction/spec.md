# Specification: User Interface & Interaction

## Goal
Build a responsive, mobile-first UI for the TheoryApp that displays questions, handles user interaction, and shows progress.

## Requirements
1.  **Layout**:
    - Mobile-first, centered layout.
    - Large, easy-to-tap buttons for answers.
2.  **Components**:
    - **Header**: Display "Mastered", "Review", and "Remaining" counts.
    - **QuestionDisplay**: Shows question text and the associated image.
    - **AnswerOptions**: A list of 4 multiple-choice buttons.
    - **Feedback**: Immediate visual feedback after an answer is selected (Green for Correct, Red for Incorrect) before moving to the next question.
3.  **Responsiveness**:
    - Ensure images fit within the screen.
    - Adjust layout for larger screens if necessary.
4.  **Transitions**:
    - Smooth transition between questions.
